Understanding Hello World

Let's go over the program again. 
        
            using System;

            namespace HelloWorld
            {
                class Program
                {
                    static void Main()
                    {
                        Console.WriteLine("Hello world!");
                    }
                }
            }
Let's find out what this code means. 
        
            using System;

This allows us to use Console.WriteLine instead of System.Console.WriteLine. (And for many more)

        
            namespace HelloWorld
            {
                // ...
            }

This isn't required, but this requires qualification or using. For example, System is a namespace.

        
            class Program
            {
                // ...
            }

All of your code must be in a class. For your code to run, the class must be called Program.

        
            static void Main(string[] args) {
                // ...
            }

This defines a static function Main where you have access to arguments and usually don't call long-running methods. We will go over what this means later, but just think of it as required code.

If you programmed in VB.NET before, here's the code in VB:

        
            Imports System
            Module HelloWorld
                Sub Main()
                    Console.WriteLine("Hello World")
                End Sub
            End Module

I won't tell you what maps to what, you figure it out yourself as an exercise.

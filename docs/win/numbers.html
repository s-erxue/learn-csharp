Data Types - Numbers

There are many ways to store numbers. Create a new project called DataTypes.

Integers

int (System.Int32)

The int data type stores integers in the range, which is really big. If you need to store integers out of the range, you can use long or System.Numerics.BigInteger (if you referenced it or used the NuGet package) Type the following:

        
            int myInt = 42;
            Console.WriteLine(myInt);

Make sure to also type at the beginning using System; just like Hello World when printing. From now on, you will have to do this yourself.

long (System.Int64)

The long type is similar to int, but it can store bigger and smaller numbers.

        
            long myLong = 2147483648L;
            Console.WriteLine(myLong);

Decimals

float (System.Single)

The float type is used to store decimal numbers from 6-7 decimal places. Type the following:

        
            float pi = 3.14F;
            Console.WriteLine(pi);

double (System.Double)

double is just like float, but with 15 digits of precision

        
            double piAsDouble = 3.14159265D;
            Console.WriteLine(piAsDouble);

Variables

Many computers need to store things. Create a new project called BasicVariables and type the following code:

        
            int myInt = 42;

This stores the value 42 in myInt. You can think of variables as a box:

We can also reassign a variable. The old value is discarded and replaced with the new value:

Let's try it out:

        
            myInt = 123;

We can even assign the value of a variable to another one:

Variables are useless and a waste of memory if we can't use them. We use them with their name, without quotes:

        
            Console.WriteLine(myInt); // 123

In VB, the code will be:

        
            Imports System
            Module BasicVariables
                Sub Main()
                    Dim myInt As Integer = 42
                    myInt = 123
                    Console.WriteLine(myInt)
                End Sub
            End Module

Data Types

The computer can only store 1s and 0s (binary), but it can store useful information like numbers. You have to tell VS2019 what you want to store. Here are a few data types:

Name Description
int Integer from -2^31 to 2^31-1
long Integer from -2^63 to 2^63-1
float Stores terminating decimals
double Stores terminating decimals with better precision
bool Stores 1 or 0 (true or false)
char Stores a character
string Stores text

We will only go through the data types this lesson, you will learn about them later. Remember when we said you can specify the data type? Here is an example:

        
            char a = 'a';
            var pi = 3.14; // var means infer type

Comments

Programmers include notes, or comments in their code. Let's see how it's done:

        
            // This is a comment
            
            /*
            This
            is
            a
            multiline
            comment
            */

In VB, this would be:

        
            ' This is a comment

            ' This
            ' is
            ' a
            ' multiline
            ' comment

Except that there are no multiline comments.

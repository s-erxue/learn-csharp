All of your code goes in a project. Let's create one.

Windows has .NET Framework, so let's use it. Use the filters C#, Windows, Console, and choose Console App (.NET Framework)

The name will be HelloWorld. Make a folder for this tutorial. Create the project

Put your cursor onto line 1, anywhere in there, and then press Alt + Enter, then Enter to remove the usings.

Add 2 blank lines at the beginning. In line one, type:

        
            using System;

At line 8 press Enter and type:

        
            Console.WriteLine("Hello world!");
